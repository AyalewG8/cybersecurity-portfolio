# Cloud Security Lab

## Project Summary

This planned lab explores security responsibilities and controls across public, private, community, and hybrid cloud environments.

## Planned Focus Areas

- Shared responsibility model
- Cloud identity and least privilege
- Secure storage and encryption
- Network segmentation and security groups
- Centralized logging and monitoring
- Backup, recovery, and resilience
- Secure configuration and compliance checks

## Planned Deliverables

- Cloud architecture diagram
- IAM role and permission matrix
- Configuration review checklist
- Logging and alerting plan
- Risk and remediation summary

## Skills Demonstrated

Cloud security · IAM · Encryption · Monitoring · Configuration management · Resilience
