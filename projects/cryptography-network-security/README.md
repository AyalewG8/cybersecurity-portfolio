# Cryptography and Network Security

## Project Summary

This project explains how cryptographic controls protect data confidentiality, integrity, authentication, and nonrepudiation across modern networks.

## Topics Covered

- Symmetric encryption: AES, DES, and 3DES
- Block cipher modes: CBC, CTR, GCM, and XTS
- Public-key cryptography: RSA, Diffie-Hellman, ElGamal, and ECC
- Hashing and message authentication: SHA-2, SHA-3, HMAC, and CMAC
- Digital certificates, certificate authorities, OCSP, and PKI
- Secure key generation, exchange, storage, rotation, and revocation
- Wireless security, secure email, and network attack mitigation

## Security Architecture

The proposed architecture combines:

- TLS for data in transit
- Strong encryption for sensitive stored data
- PKI-backed identity verification
- Centralized IAM and least privilege
- Network segmentation and secure wireless access
- Logging, monitoring, and incident response

## Deliverables

- Cryptography comparison guide
- AES and RSA process explanations
- PKI and certificate lifecycle diagram
- Secure network architecture
- Threat and mitigation analysis
- Final technical report and presentation
